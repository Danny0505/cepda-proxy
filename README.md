# Cepeda Proxy Launcher

Launcher personalizado para Amazon Fire TV Stick con estilo Dragon Ball.

## Caracteristicas
- Solo muestra apps instaladas por el usuario
- Paleta de colores Dragon Ball
- Icono: gatito con sombrero vaquero
- Soporte: sasuke_32@gmail.com

## Instalacion
1. Compila el APK (ver GitHub Actions)
2. Instala en Fire TV via ADB:
   adb install cepeda-proxy-release.apk
